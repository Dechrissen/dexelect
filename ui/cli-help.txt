Help
----

Dexelect — Progression-aware Pokemon team generator

Usage:
  ENTER – Generate a team
  G – Open the Supported Games menu to change the game mode
  M – Open the Generation Mode menu to change the party generation mode
  P – Open the Set Party Size menu to set party size (1–6)
  R – Reload the current configuration file from disk (if changes were made while the app is running)
  H – Display this help
  Q – Quit the app

Modes:
  Progression — Generated team adheres to logical game progression and config settings
  Random (Obtainable) — Generated team is completely random drawing from current game's set of obtainable Pokemon (config settings ignored)
  Random (National Dex) — Generated team is completely random drawing from current game's National Pokedex (config settings ignored)

Definitions:
  Sphere – Broad chunk of game progression (e.g. Sphere 1 for Kanto spans Pallet Town to Cerulean City)
  Distribution – Breakdown of how many party members appear in each sphere
  Lean – Semantic tag encoding the general trend in the Distribution (early_game_heavy / balanced / late_game_heavy)
  Spread – Semantic tag encoding the distance between all party members (clustered / mixed_spread / wide_spread)
  Pattern – Semantic tag encoding the pattern that the Distribution follows, if there is one (early_late_split, middle_only, dual_cluster, single_cluster)

Notes:
  - Configuration files are located in /config (e.g. /config/config_gen2.yaml for Gen 2 games)
